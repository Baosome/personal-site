# Personal Portfolio Website

A modern, responsive personal portfolio website designed to showcase my software engineering and cybersecurity projects, skills, and professional experience. Built with performance and scalability in mind, this application serves as a dynamic resume and a demonstration of modern web development practices.

![Technologies](https://skillicons.dev/icons?i=react,ts,vite,tailwind,html,css)

## Tools & Technologies

This project was built using the following modern web technologies:

*   **React 19**: For building a component-based, interactive user interface.
*   **TypeScript**: To ensure type safety, improve code quality, and enhance developer experience.
*   **Vite**: For next-generation frontend tooling, offering instant server start and lightning-fast HMR.
*   **Tailwind CSS**: A utility-first CSS framework for rapid, responsive, and consistent styling.
*   **React Router**: For seamless, client-side navigation between site sections.
*   **GitHub Pages**: For automated and reliable static site hosting.
*   **ESLint**: for static code analysis to spot problems in the code.

## Key Achievements

*   **Component Architecture**: To efficiently manage a diverse portfolio of projects (Situation), I architected a modular React application using functional components and TypeScript (Action), resulting in a maintainable reusable codebase that allows for rapid feature additions and 100% type safety (Result).

*   **Responsive Design**: Aiming to reach users on all devices (Task), I implemented a responsive layout using mobile-first design principles with Tailwind CSS (Action), achieving a consistent and visually engaging experience that retains full functionality across mobile, tablet, and desktop screens (Result).

*   **Efficient Deployment Strategy**: To streamline the release process and ensure high availability (Task), I configured a continuous deployment pipeline using GitHub Actions and GitHub Pages (Action), reducing deployment time to minutes and ensuring the latest changes are always instantly available to visitors (Result).

*   **Performance Optimization**: To ensure instant page loads and smooth interactions for users (Task), I utilized Vite's modern build tooling and optimized asset delivery (Action), eliminating build bottlenecks and achieving sub-second load times for an optimal user experience (Result).
